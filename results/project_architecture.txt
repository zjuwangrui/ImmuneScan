────────────────────────────────────────────────────────────────────
流水线（agent.py 中 LLM 通过 tool_use 顺序调用）：
────────────────────────────────────────────────────────────────────
  Input (data/input.json: HLA + mutations)
       │
       ▼
  1. generate_candidates   →  mutation_processor.py    （9-mer 滑窗）
       │
       ▼
  2. score_with_dl         →  dl_scorer.py             （AttABseq → ddG）
       │
       ▼
  3. optimize_with_mcmc    →  mcmc_optimizer.py        （ImmuneAI → MCMC loss）
       │
       ▼
  4. validate_hla_binding  →  netmhc_validator.py      （mhcflurry → IC50/%Rank）
       │
       ▼
  _rank_candidates()  综合打分：
      composite = 0.4·norm(|ddG|) + 0.3·norm(1/MCMC_loss) + 0.3·norm(1/%Rank)
       │
       ▼
  report_writer.write_report()  →  data/output/reports/<ts>_report.md + .json
       │
       ▼
  scripts/visualize.py          →  docs/figures/<ts>/*.png


────────────────────────────────────────────────────────────────────
运行方式：
────────────────────────────────────────────────────────────────────
  # 1. 安装依赖
  pip install -r requirements.txt
  mhcflurry-downloads fetch

  # 2. 设置 LLM API Key
  export DASHSCOPE_API_KEY=<your_key>

  # 3. 编辑输入（HLA + 突变）
  vim data/input.json

  # 4. 运行 Agent（生成 Top-10 报告）
  python agent/run.py

  # 5. 生成可视化图表（自动读取最新 *_results.json）
  python scripts/visualize.py
